# chem-annotation-toolkit

# A case study and open annotation guideline framework for labeling chemistry-related AI training data. This project documents how a chemistry SME would approach annotating AI-generated responses across four chemistry domains: general chemistry, organic reactions, regulatory compliance data, and industrial lubrication science.

## Purpose
This toolkit ptrovides annotation guidelines and sample labeled data for training and evaluating AI models in chemistry domains. Built by a 20+ year chemistry professional for use as a reference framework.

## Annotation Dimensions Covered
- **Scientific Accuracy** - Is the chemistry factually correct?
- **Completeness** - Are key concepts adeqately explained?
- **Clarity** - Is the explanation accessible to the target learner?
- **Regulatory Alignment** - Does content correctly reference FDA/EPA/NSF rules?
- **Safety Compliance** - Are hazards, PPE, and handling notes accurate?

## Sample Annotation Task (General Chemistry)
AI Response: "NaOH dissolves in water to form a basic solution with pH > 7."
Score: 4/5 - Accurate but omits exothermic dissolution and safety warning.
Annotation Note: Add hazard context; strong base causes burns.
